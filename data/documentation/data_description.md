# Data Description

## Data Sources
This analysis uses publicly available PM-JAY Health Benefit Package rates and market-reported private hospital prices for selected cardiac procedures in India.

## Variables
- procedure_name: Name of the cardiac procedure
- PM-JAY Price (₹): Average PM-JAY package rate
- Private Price (₹): Average private hospital price
- Price Gap (₹): Difference between private and PM-JAY prices
- Price Ratio: Ratio of private price to PM-JAY price

## Data Characteristics
- Aggregated, non-patient-level data
- No personally identifiable information
- Suitable for policy and health economics analysis
